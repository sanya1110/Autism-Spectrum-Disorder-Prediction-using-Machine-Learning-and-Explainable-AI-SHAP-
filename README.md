🧠 Autism Spectrum Disorder Prediction using Machine Learning & SHAP

## 📌 Project Overview

This project focuses on developing an intelligent and interpretable system for predicting Autism Spectrum Disorder (ASD) using Machine Learning techniques. The system analyzes behavioral, demographic, and screening data to estimate the likelihood of autism and supports early screening.

A key feature of this project is the integration of **Explainable AI (XAI)** using **SHAP**, which provides clear explanations of model predictions, making the system transparent and trustworthy.


## 🎯 Problem Statement

Traditional methods for diagnosing autism are time-consuming, require expert professionals, and may not be easily accessible in all regions. This often leads to delayed diagnosis and intervention.

This project aims to provide a **fast, data-driven, and accessible screening tool** to assist in early identification of ASD.


## 🎯 Objectives

* To build and train machine learning models for accurate ASD prediction
* To assist in early screening using behavioral and demographic data
* To apply explainability (SHAP) for transparent and interpretable predictions
* To identify key features influencing autism prediction

## 🔄 Methodology

### 1. Data Collection

* Dataset obtained from publicly available sources (e.g., Kaggle)

### 2. Data Preprocessing

* Handling missing values
* Label encoding of categorical features
* Outlier detection using IQR method
* Class imbalance handling using SMOTE

### 3. Exploratory Data Analysis (EDA)

* Distribution plots
* Box plots
* Correlation heatmaps
* Feature analysis

### 4. Model Training

* Logistic Regression
* Decision Tree
* Random Forest
* Support Vector Machine
* XGBoost

### 5. Model Evaluation

* Accuracy
* Precision
* Recall
* F1-score
* Confusion Matrix

### 6. Explainability (XAI)

* SHAP used to:

  * Explain feature importance
  * Provide local and global interpretations
  * Visualize model decisions
